Bio

I am a Self-taught Web Developer who lives in the Pretoria area. I started coding when I was in highschool and I have continued to code as a hobbyist outside of highschool. In the last several months I have been reading up everything I can that is pertaining to Web development and equiping myself with marketable skills.

My goal is to become a Full-Stack developer in the coming years. So my focus is on both the backend and front-end. I don't really see a difference between creating a database with some SQL code or creating a cool looking button because I have this thing where if I see code, I need to understand what it does and how it does it. While growing as a developer, I have no shame in reaching out for help when Google searches just aren't being as helpful as they could be. I also enjoy sharing the things I may happen to know if a team member needs some help.

I am a driven individual who prides himself on being self-taught. So if there are frameworks/libraries that are specific to your development process that I am not familiar with, all I need is a few days to familiarise myself with the basics of that framework/libraries.
